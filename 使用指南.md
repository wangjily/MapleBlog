## 发表文章

### 博客文章（blog）

**位置：** `src/content/blog/`

**文件命名：** `文章标题.md`

**Frontmatter 格式：**

```markdown
---
title: 文章标题
description: 文章简介，一句话描述
createdAt: 2025-01-15
tags:
  - 标签1
  - 标签2
categories:
  - 分类1
featured: true
status: published
---

正文内容，使用 Markdown 格式...
```

**字段说明：**

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| title | 是 | 文章标题 | "我的第一篇文章" |
| description | 是 | 文章简介 | "这是一篇介绍..." |
| createdAt | 是 | 创建日期 | "2025-01-15" |
| tags | 否 | 标签列表 | ["技术", "生活"] |
| categories | 否 | 分类列表 | ["随想"] |
| featured | 否 | 是否推荐 | true/false |
| status | 否 | 发布状态 | published/draft |

**完整示例：**

```markdown
---
title: 跨越星空彼岸，逆转时间长河
description: 一段跨越星空的旅程
createdAt: 2026-05-04
tags:
  - 星空
  - 科幻
  - 归途
categories:
  - 随想
featured: true
status: published
---

在无尽的星海中，我曾是一个迷失的旅人。

跨越星空彼岸，穿过无数光年的距离...

---
```

### 5. 友链配置（friends.json）

**位置：** `public/data/friends.json`

**JSON 格式：**

```json
{
  "categories": [
    { "id": "cat-tech", "name": "技术博客", "code": "tech", "sort_order": 1 },
    { "id": "cat-life", "name": "生活记录", "code": "life", "sort_order": 2 }
  ],
  "links": [
    {
      "id": "site-001",
      "name": "站点名称",
      "url": "https://example.com",
      "description": "一句话简介",
      "avatar": "https://example.com/avatar.png",
      "status": "active",
      "category_code": "tech",
      "sort_order": 1
    }
  ]
}
```

**字段说明：**

| 字段 | 说明 | 可选值 |
|------|------|--------|
| status | 友链状态 | `active`（正常）、`error`（异常）、`lost`（失效不显示） |
| sort_order | 排序，数字越小越靠前 | 整数 |

### 6. 网站配置（config.ts）

**位置：** `src/lib/config.ts`

**主要配置项：**

```typescript
// 网站信息
export const SITE_INFO = {
  NAME: '网站名称',
  SITE_NAME: '网站名称',
  SUBNAME: '网站副标题',
  DESCRIPTION: '网站描述',
  URL: 'https://你的域名.com',
  AUTHOR: '作者名称',
  START_DATE: '2026-05-04',  // 网站上线日期
  ICP: {
    NUMBER: '备案号',
    URL: '备案链接'
  }
} as const;

// 分页配置
export const PAGINATION_CONFIG = {
  BLOG_ENTRIES_PER_PAGE: 10,  // 博客每页数量
  NOTES_DEFAULT_PAGE_SIZE: 10, // 笔记每页数量
} as const;
```

| 变量名 | 说明 | 可选 |
|--------|------|------|

```bash

PUBLIC_BAIDU_ANALYTICS_ID=xxxxxxxxxx
```

### 9. 首页图片轮播

**位置：** `src/components/home/ImageCarousel.astro`

**功能特性：**
- 自动轮播（5秒切换）
- 左右箭头手动切换
- 底部导航指示点
- 播放/暂停按钮
- 键盘导航（方向键切换，空格暂停）
- 触摸滑动支持（移动端）
- 鼠标悬停暂停

**图片配置：** 在组件中修改 `carouselImages` 数组即可更换图片。

## 构建与部署

### 1. 构建网站
```bash
pnpm run build
```

构建完成后，`dist` 文件夹包含所有静态文件。

### 2. 部署到服务器

**宝塔面板方式：**
1. 打开宝塔 → 网站 → 文件管理器
2. 进入网站根目录
3. 删除旧的 `assets` 等文件夹
4. 上传 `dist` 里的所有内容

**FTP 方式：**
1. 用 FileZilla 连接服务器
2. 将 `dist` 内容拖拽上传

## 常见问题

### Q: 新建文章后不显示？
1. 确认文件在正确目录（`src/content/blog/`）
2. 确认文件名是 `.md` 结尾
3. 确认 Frontmatter 格式正确
4. 运行 `pnpm run build` 重新构建

### Q: 图片不显示？
1. 图片必须放在 `public/assets/uploads/` 目录
2. 引用路径要用 `/assets/uploads/图片名.jpg`

### Q: 如何删除文章？
直接删除对应的 `.md` 文件，然后重新构建上传。

### Q: 如何修改网站标题？
修改 `src/lib/config.ts` 中的 `SITE_INFO` 配置。

### Q: 如何添加视频？
1. 把视频文件放到 `public/videos/` 目录
2. 在文章中使用 `<video>` 标签引用

### Q: 友链状态说明？
- `active` - 正常显示
- `error` - 显示红色错误标记
- `lost` - 不显示（已失效）

### Q: 如何自定义首页轮播图片？
编辑 `src/components/home/ImageCarousel.astro` 文件中的 `carouselImages` 数组。

## 优化说明

### 安全优化
- XSS 防护：所有用户输入均经过 HTML 转义和 URL 验证
- 敏感信息隔离：Analytics ID 移至环境变量

### SEO 优化
- HTML 语言：默认设置为 `zh-CN`
- 结构化数据：文章页面添加 Article 和 BreadcrumbList JSON-LD
- Sitemap：包含分类、标签等完整页面

### 代码质量
- 移除所有生产环境调试代码
- 完善可访问性（ARIA 标签、键盘导航）

### 用户体验
- 搜索功能加载状态提示
- 移动端菜单 ESC 键关闭
